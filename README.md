Sales Dashboard (Excel)

Project Overview
This project presents an interactive Sales Dashboard built in Microsoft Excel to analyze business performance, revenue trends, and customer insights.
The dashboard provides a clear and visual representation of key sales metrics to support data-driven decision making.

Tools & Skills Used
* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Data Visualization
* Dashboard Design

Dataset Information
The dataset includes:
* Sales transactions
* Revenue data
* Customer information
* Product categories
* Order dates

 Dashboard Features
* Sales trend over time
* Product category performance
* Customer segmentation
* Revenue analysis
* Region-wise sales performance

Dashboard Preview

Key Insights
* Identified top-performing products
* Highlighted high-revenue regions
* Analyzed monthly sales trends
* Found customer purchase patterns
